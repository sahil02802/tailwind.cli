###Git Initilization : git init
###Push to stage area : git add .
###Commit : git commit -m "Commit_msg"
###Connect to main branch : git branch -M main
###Create Branch : git branch branch_name , git checkout branch_name
###Switch Bramch : git switch branch_name
###Create and switch branch : git switch -c branch_name, git checkout -b branch_name
###Connect to remote Directory : git remote origin add <Remote_Repo_URL>  
###Push in Remote repo : git push -u origin main
###Connect your github ac name to git : git config --global user.name "Your_Github_Ac_Name", git config --local user.name "Your_Github_Ac_Name"  
###Connect your github ac email to git : git config --global user.email "Your_Github_Ac_email", git config --local user.email "Your_Github_Ac_email"
###check your linked status : git config --list
###chekc status : git status
###Show Repo URL (Connected or not) : git remote -v
###unset account name : git config --global --unset user.name
###unset account email : git config --global --unset user.email
